# Requirements

## Software Requirements

- Node.js 20 or later
- npm 10 or later
- A modern desktop browser such as Google Chrome, Microsoft Edge, or Firefox

## Project Dependencies

Main runtime libraries:

- React
- React DOM
- React Router DOM
- Recharts
- Axios

Main development libraries:

- TypeScript
- Vite
- Tailwind CSS
- ESLint

## Recommended Environment

- Operating System: Windows 10 or Windows 11
- Terminal: PowerShell or Command Prompt
- Editor: Visual Studio Code

## Run Requirements

Before running the project:

1. Open the `frontend` folder
2. Install dependencies with `npm install`
3. Start the app with `npm run dev`

## Build Requirements

To verify the project before submission:

```powershell
cd frontend
npx tsc -b
npm run build
```

## Input File Requirement

If you want to test scenario import, use:

- `scenario_import_template.json`

This file contains a sample intersection scenario for geometry, traffic, and signal data.
